Simplesmente Você

Não tem como escrever você. Não tenho a capacidade de colocar você em poucas palavras uma definição. Para os poetas, você seria um papel branco. Para os pintores, você seria uma tela em branco. Para fotógrafos, você seria um lugar. Viu como isso é tudo raso? Colocar apenas uma definição? Para os poetas, você seria os pensamentos colocados em um simples papel, mas quando se descreve alguém tão inspirador como você, o papel se transforma em um sentimento. Para os pintores, você é a musa incompreendida, colocado em telas para não ser transbordar nos pensamentos, para mostrar em pintura como você é. Para os fotógrafos, você é inspirador, um ser único que temos que fotografar para deixar registrado. Você não é de menos para deixar passar.
